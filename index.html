About

Keep In Time or KIT is a online web tool used to create simple, stress free, and robust project management charts. Whether you are a software engineering student and you need to manage your coursework project or whether you are a software project manager and you need robust and complex management software to help your project run on time; KIT is a tool intended for use for small one person projects, to large teamwork business critical projects.

Create project charts:

WBT

Create your own work break down tree.

Quickly and easily see what has to be done.

A work break down tree (WBT) or work breakdown structure (WBS) is a chart in which the critical work elements, called tasks, of a project are illustrated to portray their relationships to each other and to the project as a whole.

PERT

Create your own Pert chart.

Put things in order of when they should be done.

A PERT chart presents a graphic illustration of a project as a network diagram consisting of numbered nodes (either circles or rectangles) representing events, or milestones in the project linked by labelled vectors (directional lines) representing tasks in the project.

GANTT

Create your own GANTT chart.

Plan your project by creating a detailed schedule

Frequently used in project management, a Gantt chart provides a graphical illustration of a schedule that helps to plan, coordinate, and track specific tasks in a project.
